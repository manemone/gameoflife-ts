# lifegame-ts
## What's this?
An implementation of [Conway's game of life](https://en.wikipedia.org/wiki/Conway%27s_Game_of_Life) for learning TypeScript.

## Dependencies
- npm
- webpack

## Setting up
```bash
$ npm install
```

## Scripts
### Build
#### For development environment
```bash
$ npm run build-dev
```

#### For prod environment
```bash
$ npm run build-prod
``` 

### Cleaning up build directory
```bash
$ npm run clean
```

### Starting dev server
```bash
$ npm run start
```
